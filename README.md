

# University Projects Archive

This repository serves as a professional portfolio documenting my academic and technical journey as a Computer Engineering student at Benha University. It serves as a comprehensive archive of both individual assignments and collaborative team-based projects developed throughout my undergraduate years.

## Repository Overview

This collection tracks my evolution from foundational engineering principles to advanced software and systems implementation. It highlights my technical versatility across various domains, including embedded systems, software architecture, machine learning, and hardware design.

## Technical Scope

My work across these projects covers several core engineering fields:

* **Software Engineering:** Clean Architecture, MVVM, and BLoC/Cubit state management.
* **Computer Science Foundations:** Operating Systems, Computer Architecture, and Data Structures & Algorithms.
* **AI & Advanced Systems:** Machine Learning, Computer Vision, and RAG-integrated applications.
* **Embedded & Hardware:** Logic Circuits, PCB design, and semiconductor project development.

## Project Structure

The repository is organized by academic progression to demonstrate the development of my technical proficiency:

* **Year 01 - 02:** Foundational projects including logic circuits, audio amplifiers, and initial web development.
* **Year 03:** Core discipline projects covering AI, databases, networking, embedded systems, and computer architecture.
* **Final Year:** Advanced team-based capstone projects, including specialized implementations in Operating Systems, Machine Learning, and Computer Vision.

## Methodology & Collaboration

Many of the projects listed in the "Final Year" and "Projects 3" folders represent intensive team collaborations. These projects emphasize my experience in version control, feature allocation, and technical team leadership. All codebases are documented to reflect maintainable, scalable, and high-quality software engineering standards.
